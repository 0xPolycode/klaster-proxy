== Get Wallet Activity API

.Request Example
[%collapsible]
====
```
GET /api/get-activity?wallet=0x000000000000000000000000000000000000000a HTTP/1.1
Host: localhost:8080
```
====
.Response Example
[%collapsible]
====
```
HTTP/1.1 200 OK
Content-Type: application/json

{
    "ccip_api_responses": [
        { ... } // same structure as Klaster API response
    ]
}
```
